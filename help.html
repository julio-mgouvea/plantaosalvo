Como Usar o Sistema

1. Acesso ao Sistema

Para acessar o sistema, você precisa criar uma conta ou fazer login com suas credenciais existentes:

  • Criar conta: Clique em "Criar Conta" na tela de login, preencha um nome de usuário e senha, e confirme a senha.
  • Fazer login: Insira seu nome de usuário e senha nos campos correspondentes e clique em "Entrar".

2. Cadastro de Pacientes

Para cadastrar um novo paciente:

  1. Preencha o formulário com os dados do paciente (nome, prontuário, hipótese diagnóstica, condutas realizadas)
  2. Selecione a prioridade de atendimento (Alta, Média ou Baixa)
  3. Registre as pendências, se houver
  4. Clique em "Adicionar Paciente"

Observação: O campo de prontuário é opcional.

3. Gerenciamento de Pacientes

Na lista de pacientes, você pode:

  • Visualizar detalhes: Clique no botão "Ver" para visualizar todas as informações do paciente
  • Remover paciente: Clique no botão "Remover" para excluir um paciente da lista
  • Filtrar por prioridade: Use o seletor de prioridade para filtrar pacientes por nível de urgência

4. Geração de Relatório

Para gerar um relatório de passagem de plantão:

  1. Preencha a data do plantão (obrigatório)
  2. Opcionalmente, preencha o nome do médico plantonista e do próximo plantonista
  3. Adicione observações gerais, se necessário
  4. Clique em "Gerar Relatório"

Para imprimir o relatório, clique no botão "Imprimir Relatório". Uma nova janela será aberta com uma versão otimizada para impressão.

5. Exclusão de Dados

Para excluir todos os dados de pacientes manualmente:

  1. Clique no botão "Excluir Todos os Dados" na parte inferior da página
  2. Confirme a ação quando solicitado

Observação: Todos os dados são automaticamente excluídos após 24 horas por questões de privacidade.

Privacidade do Paciente e Conformidade com a LGPD

Este sistema foi desenvolvido com foco na proteção de dados dos pacientes e em conformidade com a Lei Geral de Proteção de Dados (LGPD).

Medidas de Proteção de Dados

1. Autenticação de Usuários

O acesso ao sistema é protegido por autenticação com usuário e senha, garantindo que apenas profissionais autorizados possam acessar as informações dos pacientes.

2. Exclusão Automática

Todos os dados dos pacientes são automaticamente excluídos após 24 horas, minimizando o risco de exposição prolongada de informações sensíveis.

3. Exclusão Manual

O sistema oferece a opção de excluir todos os dados imediatamente após a passagem de plantão, permitindo que você controle quando as informações são removidas.

4. Armazenamento Local

Os dados são armazenados localmente no navegador do usuário, não sendo enviados para servidores externos, o que aumenta a segurança e privacidade.

Princípios da LGPD Aplicados

  • Finalidade: Os dados são coletados apenas para o propósito específico de facilitar a passagem de plantão médico.
  • Adequação: Apenas informações relevantes para o atendimento médico são registradas.
  • Necessidade: A coleta é limitada ao mínimo necessário para a finalidade pretendida.
  • Livre acesso: Os dados podem ser facilmente visualizados pelos profissionais autorizados.
  • Qualidade dos dados: O sistema permite a atualização e correção das informações.
  • Transparência: Informações claras sobre o tratamento dos dados são fornecidas aos usuários.
  • Segurança: Medidas técnicas e administrativas são aplicadas para proteger os dados pessoais.
  • Prevenção: O sistema adota medidas para prevenir danos aos titulares dos dados.
  • Não discriminação: Os dados não são utilizados para fins discriminatórios.
  • Responsabilização: O sistema foi projetado para demonstrar a conformidade com as normas de proteção de dados.

Recomendações de Boas Práticas

  1. Utilize o sistema apenas em dispositivos seguros e de acesso restrito.
  2. Não compartilhe suas credenciais de acesso com outras pessoas.
  3. Faça logout ao finalizar o uso do sistema, especialmente em computadores compartilhados.
  4. Utilize a função de exclusão manual após a passagem de plantão ser concluída.
  5. Evite incluir informações de identificação desnecessárias nos registros.
  6. Ao imprimir relatórios, garanta que sejam armazenados de forma segura e descartados adequadamente após o uso.
